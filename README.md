# Teste Técnico Front-end - Econverse 🚀

Este repositório contém o desenvolvimento de uma landing page de e-commerce para o desafio técnico da **Econverse**. O objetivo principal foi transformar um design do Figma em uma aplicação funcional, performática e responsiva, utilizando tecnologias modernas de desenvolvimento web.

## 🛠️ Tecnologias Utilizadas
- **React** (Vite)
- **TypeScript**
- **SASS / CSS Modules**
- **Git & GitHub**

## ✨ Funcionalidades Implementadas
- **Vitrine Dinâmica:** Carrossel de produtos com navegação funcional por setas.
- **Modal de Produto:** Interface detalhada em duas colunas (imagem e informações), com seletor de quantidade e botão de compra.
- **Consumo de API:** Integração para carregamento dinâmico de dados dos produtos.
- **Responsividade:** Layout adaptado para diferentes resoluções de tela.
