# Teste Front-end - Econverse 🚀

Este repositório contém o desenvolvimento de uma landing page de e-commerce para o desafio técnico da **Econverse**. O objetivo foi transformar um design do Figma em uma aplicação funcional utilizando React, TypeScript e SASS.

## 👤 Autora
**Jéssica dos Santos Fernandes**
*Desenvolvedora em transição de carreira da Biomedicina para a Tecnologia.*

## 🛠️ Tecnologias Utilizadas
- **React** (Vite)
- **TypeScript**
- **SASS / CSS Modules**

## ✨ Funcionalidades Principais
- **Vitrine de Produtos:** Carrossel dinâmico com navegação por setas.
- **Modal de Compra:** Layout detalhado com seletor de quantidade e informações do produto.
- **Responsividade:** Interface adaptada para diferentes dispositivos.
